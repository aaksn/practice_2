Описание проекта

Ввод и отображение информации о посещаемости студентов

Описание

В качестве темы для проекта был выбран ввод и отображение информации о посещаемости студентов. Он будет позволять редактировать таблицу посещаемости, импортировать её из excel и экспортировать в excel или pdf.

Работа над проектом будет вестись двумя студентами: Антоном Аксеновым и Русланом Погореловым

Функционал

Проект будет содержать реализацию следующих функций:

  • Разграничение прав пользователей, их авторизация
  • Просмотр таблицы с посещаемостью
  • Редактирование посещаемости
  • Редактирование списка студентов и групп
  • Импорт файла excel
  • Экспорт в формате excel
  • Экспорт в формате pdf

Ответственность за реализацию функционала

  • Разграничение прав пользователей, их авторизация - Погорелов Руслан
  • Вёрстка макетов страниц - Аксенов Антон
  • Разработка отправки запросов на сервер для редактирования посещаемости, списка студентов и групп - Аксенов Антон
  • Работа с базами данных на серверной стороне - Погорелов Руслан
  • Импорт файла excel - Аксенов Антон
  • Экспорт в формате excel - Аксенов Антон
  • Экспорт в формате pdf - Погорелов Руслан

Технологии

В качестве технологий для проекта будут использоваться следующие компоненты:

  • HTML и CSS для отображения контента на клиентской стороне (в браузере)
  • PHP язык программирования для серверной стороны
  • JavaScript для программирования интерактивных элементов на клиентской стороне (в браузере)
  • Apache HTTP Server в качестве Web-сервера

Этапы разработки

  1. Разработка и вёрстка макетов страницы
  2. Настройка программного обеспечения на сервере, создание базы данных
  3. Создание таблиц в базе данных
  4. Содание системы импорта и экспорта
  5. Реализация редактирования посещаемости и списка
